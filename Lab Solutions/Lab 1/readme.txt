This is an example of Lab1 exercise solution.

1. You need to go into ScalableMachineLearning directory, which has been downloaded from GitHub.

2. Place the Lab_1_exercise.py file in the Code folder, and replace 'your_username' with your actual username in the SparkSession builder configuration at the beginning of the PySpark program.

3. Put Lab1_exercise.sh into HPC folder

4. Then run 'sbatch ./HPC/Lab_1_exercise.sh' under /users/$USER/com6012/ScalableML directory

5. Output file will be appearing in the Output folder.
